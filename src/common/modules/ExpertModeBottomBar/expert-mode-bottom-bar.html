The modulus length must be a multiple of 8 and >= 256 and <= 16384. A larger modulus length will generate longer keys and allow for longer messages. A smaller hash will allow for longer messages. SHA-1 is considered insecure and should no longer be used. Longer keys will take a long time to generate.
